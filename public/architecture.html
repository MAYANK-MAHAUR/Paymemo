PayMemo · Architecture

How it works

PayMemo is the private memory layer for every Morph transaction. Memos are encrypted in your browser before they ever leave the device — the server only ever sees ciphertext.

Trust model

Server is treated as untrusted. Even a full database dump leaks no memo text — only ciphertext, public chain data, and salts.

Key derivation

The AES key is derived from a wallet personal_sign over a fixed PayMemo message. No password to lose, no key on the server.

Real-time path

Railway worker is the hot path (~2s). Vercel daily cron + page-load catch-up keeps it correct if Railway is down.
